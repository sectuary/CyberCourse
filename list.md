## **Penetration Testing (General)**

### **Beginner Level**

1. **[eLearnSecurity Junior Penetration Tester (eJPT)](https://ine.com/security/certifications/ejpt-certification)**  
   - **Price:** **$199** (Exam voucher only). Often bundled with training.  
   - **Focus**: Foundational penetration testing, networking, and security tools.  
   - **Registration**: Purchase on the INE website.  
   - **Course Overview**: This certification introduces penetration testing fundamentals, including TCP/IP networking, information gathering, scanning, and exploitation basics. It emphasizes practical use of security tools like Nmap, Metasploit, and Burp Suite. Ideal for beginners, it provides hands-on experience to build confidence in assessing vulnerabilities and conducting ethical hacking tasks.  

2. **[Practical Junior Penetration Tester (PJPT)](https://certifications.tcm-sec.com/pjpt/)**

   - **Registration**: Sign up and purchase on the TCM Security website.

The exam simulates real-world scenarios, requiring candidates to demonstrate their ability to compromise systems and submit a professional penetration testing report.

   - **Registration**: Not a certification; public framework.

## **Certification Path Recommendations**

### **For Beginners Starting in Cybersecurity:**
1. CompTIA Security+ (Foundation)
2. eJPT or PJPT (Hands-on introduction)
3. Specialize based on interest (Web App, Cloud, etc.)

### **For Penetration Testing Career Path:**
1. eJPT → OSCP → OSEP/OSWE → OSCE3
2. Alternative: PJPT → eCPPT → eCPTXv2

### **For Cloud Security Specialization:**
1. Cloud Fundamentals (AWS/Azure/GCP) → CCSP → Cloud-specific security certs
2. For offensive: GCPN or cloud-focused penetration testing

### **For Red Team Operations:**
1. OSCP foundation → CRTP → CRTO → CRTE/CRTM
2. Alternative: GPEN → GRTP → Advanced red team certs

### **For Web Application Security:**
1. eJPT foundation → eWPT → Burp Suite Certified → OSWE/eWPTXv2
2. Bug bounty focus: CBBH pathway
